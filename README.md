# UE4-24-3_SourceCode
此目录为引擎根目录/Engine目录下  
分支说明:  
- develop:基于原始引擎代码添加用于MobileSkin的自定义ShadingModel
- MobileSkin: 仅预积分贴图的移动端皮肤  
- SG_SSS: 使用高斯函数次表面散射的移动端皮肤  
以上分支都是独立基于UE4.24.3原始引擎代码的修改，并不包含彼此分支代码。
